# AI Tooling

## Tools Used

- ChatGPT
- Sentence Transformers
- OpenAI API
- ChromaDB

---

## What AI Was Used For

AI was used to:
- Design the system architecture
- Generate and structure code
- Debug errors and fix issues
- Improve evaluation logic
- Assist with documentation writing

---

## What Worked Well

- Rapid development of boilerplate code
- Debugging complex issues quickly
- Generating structured evaluation pipelines
- Improving code organization

---

## What Did Not Work Well

- Some generated code required manual fixes
- Not all outputs were immediately correct
- Required validation and testing after generation

---

## Validation Process

All AI-generated content was:
- Tested locally
- Verified through unit tests
- Checked via evaluation dataset
- Manually reviewed and corrected

---

## Conclusion

AI significantly accelerated development but required human validation to ensure correctness and quality.
