# Information Security Policy

## Purpose
To protect company data and systems from unauthorized access.

## Access Control
- Role-based access is enforced
- Passwords must be strong and changed every 90 days

## Data Protection
- Sensitive data must be encrypted
- Data sharing must follow company protocols

## Incident Reporting
All security incidents must be reported within 24 hours.

## Network Security
- VPN required for remote access
- Firewall and antivirus must be active

## Compliance
Employees must comply with all security policies and training.

## Violations
Security violations may result in disciplinary action.
